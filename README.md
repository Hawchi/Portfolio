# Paulien's Portfolio

Mijn persoonlijke portfolio, gebouwd met Vue.js en Vite.

Live at [paulienma.nl](https://paulienma.nl)

## About
Een portfolio met een hoofdpagina met info over mij, mijn skills en projecten, en voor elk project een eigen detailpagina. Ik ben een 2e jaars HBO-ICT student aan Fontys en hou van creatieve projecten waar ik mijn eigen ding in kwijt kan.

## Projects Featured
- **Border:Moonlight** — Fan website voor ENHYPEN met lore, discografie, member profiles. Gebouwd met HTML, CSS en JavaScript.
- **Digital Paradise** — Branding en website voor een elektronische artiest met Caribische roots. Gebouwd met HTML, CSS en JavaScript.
- **CZ Zorgvinder** — UX/UI redesign van de CZ Zorgvinder, gericht op toegankelijkheid en empathie voor kwetsbare gebruikers.
- **OWOW Atlas** — Centrale animatiebibliotheek voor OWOW. Gebouwd met React, GSAP, Framer Motion en CSS.
